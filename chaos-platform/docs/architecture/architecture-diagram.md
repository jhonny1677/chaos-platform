## Request Flow: Chaos Experiment

```
Developer / EventBridge
        │
        ▼ POST /experiments
  chaos-engine API
        │
        ├──► Kubernetes API ──► kill target-app pods
        │
        ├──► Kafka producer ──► topic: chaos-events
        │
        ├──► DynamoDB ──► write experiment record
        │
        └──► SNS ──► slack-notifier ──► #chaos-experiments
                                (experiment.started event)
```

## Request Flow: Report Generation

```
DynamoDB experiment result
        │
        ▼ S3 ObjectCreated (results/*.json)
  report-generator Lambda
        │
        ├──► DynamoDB GetItem ──► full experiment data
        │
        ├──► report_builder.py ──► HTML string
        │
        ├──► pdf_generator.py (WeasyPrint) ──► PDF bytes
        │
        ├──► S3 PutObject ──► s3://chaos-reports/reports/*.pdf
        │
        └──► SNS Publish ──► slack-notifier ──► #chaos-reports
                                (report.generated event with presigned URL)
```
